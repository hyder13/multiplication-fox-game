# 九九乘法闖關王 🦊

一個適合小孩的九九乘法互動練習遊戲。3 分鐘一局、衝 100 分過關，一關一關把乘法練到倒背如流。

## 玩法
- 10 個關卡：2 的乘法 → 9 的乘法 → 綜合挑戰 → 倒背高手。
- 每局 3 分鐘，衝到 100 分就過關（又快又準分數更高）。
- 過關才解鎖下一關，星星越多越厲害 ⭐。
- 進度自動存在瀏覽器，手機 / 平板 / 電腦都能玩。
