# 九九乘法闖關王 🦊

一個適合小孩的九九乘法互動練習遊戲。3 分鐘一局、衝 100 分過關，一關一關把乘法練到倒背如流。

## 玩法
- 兩種玩法，首頁可切換：
  - **九九乘法**（10 關）：2 的乘法 → 9 的乘法 → 綜合挑戰 → 倒背高手。
  - **加減法**（5 關）：1 位數加減法闖關（減法不會出現負數）。
  - **除法**（5 關）：整除練習 → 綜合除法 → 有餘數挑戰。
- 每局 3 分鐘，衝到 100 分就過關（又快又準分數更高）。
- 過關才解鎖下一關，星星越多越厲害 ⭐。
- 進度自動存在瀏覽器，手機 / 平板 / 電腦都能玩。

## 安裝到手機 / 平板
- iPhone / iPad：用 Safari 打開遊戲 → 分享 → 加入主畫面。
- Android：用 Chrome 打開遊戲 → 選單 → 加到主畫面 / 安裝應用程式。
- 安裝後可全螢幕開啟，並會快取主要圖片與音效，方便離線重複練習。
